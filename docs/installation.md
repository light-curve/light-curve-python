# Installation

```sh
pip install 'light-curve[full]'
```

The `[full]` extra adds `iminuit` and `scipy`, required for non-linear fitting features
(`BazinFit`, `VillarFit`, `RainbowFit`).

For conda:

```sh
conda install -c conda-forge light-curve-python
```

See the [README](https://github.com/light-curve/light-curve-python#installation) for
platform-specific notes and the full support matrix.
